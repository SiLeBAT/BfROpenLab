What is BfROpenLab?

BfROpenLab is a collection of several extensions for the data analytics software KNIME (http://www.knime.org/) developed at the Federal Institute for Risk Assessment in Germany (BfR). It contains the following extensions:

  • KNIME Foodchain Tracing Nodes
  • KNIME GIS Nodes
  • KNIME NLS Nodes
  • SBML

How to install it?

Download KNIME from http://www.knime.org/ and install it. Extensions to KNIME can be installed via Help -> Install New Software in the Main Window.

Press Add to add BfROpenLab repository: http://dl.bintray.com/silebat/generic/

Select the extensions you would like to install, press Next and follow the menu prompts.
